PalindromemordnilaP

(A Palindromaton)

PalindromemordnilaP is an app based on the 'string reversal' coding challenge, but with a twist!
The user will input a word or phrase, and PalindromemordnilaP will reverse the string, then compare the reversed string with the original to test for Identicality.
If the two strings are identical, the phrase is a palindrome and the user wins.
String Reversal is typically used to test a person's algorithmic understanding.

I made PalindromemordnilaP using HTML, CSS, Bootstrap, JavaScript, & jQuery.
